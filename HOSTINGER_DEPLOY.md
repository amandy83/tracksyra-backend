# Hostinger deployment

Repository Name:

```text
tracksyra-backend
```

Hostinger Framework:

```text
Express
```

Node Version:

```text
22.x
```

Root Directory:

```text
./
```

Build Command:

```text
npm install
npm run build
```

Start Command:

```text
npm start
```

Run `npm run worker` as a separate worker process/application where Hostinger supports it. Configure every production variable from `backend-production.env.example` in Hostinger's environment settings.
